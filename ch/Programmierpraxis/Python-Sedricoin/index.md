Bitcoin, Ethereum und Tether gehören derzeit zu den 
[größten](https://coinmarketcap.com/) 
sogenannten 
[Kryptowährungen](https://en.wikipedia.org/wiki/Cryptocurrency).
Alle basieren auf dem Konzept der 
[Blockchain](https://en.wikipedia.org/wiki/Blockchain) 
und ermöglichen es, ohne eine zentrale Autorität
digitales Geld sicher an andere Personen zu übertragen.

Obwohl die Bitcoin-Blockchain inzwischen so bedeutsam für die Weltwirtschaft ist,
lassen sich die wichtigsten Mechanismen mit geringem Aufwand nachbauen.
In dieser Aufgabengruppe implementieren Sie Schritt für Schritt Ihre eigene kleine
Blockchain-Kryptowährung.
Das Vorbild ist dabei Bitcoin.

👉️ Daraus ergibt sich der Name *Sedricoin*.

- In den ersten Aufgaben implementieren Sie die Grundlagen einer Blockchain:
  Blöcke verketten, Transaktionsdaten speichern und die Blockchain gegen Manipulation
  schützen.
- Achließend wird das Berechnen neuer Blöcke implementiert.
- Am Ende des Projekts wird die Anwendung so erweitert, dass ein simples
  Peer-to-Peer-Netzwerk aufgespannt werden kann.

Falls Sie sich für die theoretischen Hintergründe interessieren, können Sie hier fündig werden:

- [Bitcoin-Whitepaper](https://bitcoin.org/bitcoin.pdf)
- [Bitcoin Developer Guide](https://developer.bitcoin.org/devguide/index.html)
