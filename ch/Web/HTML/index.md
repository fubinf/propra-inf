title: HTML
---
[TERMREF::HTML], kurz für HyperText Markup Language, ist die grundlegende [TERMREF::Auszeichnungssprache]
für die Struktur und den Inhalt von Webseiten. 
HTML-Tags beschreiben die semantische Struktur von Inhalten
(z.B. Überschrift, Absatz, Aufzählungspunkt).

In der Frontend-Entwicklung spielt HTML eine entscheidende Rolle:
Es beschreibt das grundlegende Gerüst der Webseite, auf das sich dann
CSS-Regeln (für Layout) und JavaScript-Code (für Interaktivität) beziehen.
